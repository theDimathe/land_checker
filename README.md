# Landing control room

Локальная среда для просмотра и быстрой проверки лендингов в папках репозитория.

## Что умеет

- Автоматически находит все папки с лендингами в корне репозитория.
- Показывает admin-страницу (`/`) с таблицей:
  - номер;
  - имя папки;
  - превью первого экрана (iframe);
  - дата последнего изменения в папке;
  - количество файлов и общий размер;
  - количество форм и внешних скриптов;
  - есть ли `viewport` и `title`;
  - индикаторы наличия Яндекс.Метрики, Facebook Pixel, Google Tag Manager.
- Позволяет открыть любой лендинг в новой вкладке.

## Запуск

```bash
node server.js
```

По умолчанию откроется на `http://localhost:4173`.

Можно переопределить порт:

```bash
PORT=3000 node server.js
```

## Добавление новых лендингов

Просто добавьте новую папку в корень репозитория c `index.html`.
После перезагрузки страницы админки новый лендинг появится автоматически.
